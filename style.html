HTML STYLES

The HTML style attribute is used to add styles to an element, such as color, font, size, and more.

HTML Style Attribute






Property

Style Property Defines
Background Color the background color for an HTML element.
Text Color the text color for an HTML element
Fonts the font to be used for an HTML element
Text Size the text size for an HTML element
Text Alignment the horizontal text alignment for an HTML element

CHAPTER SUMMARY

  • Use the style attribute for styling HTML elements
  • Use background-color for background color
  • Use color for text colors
  • Use font-family for text fonts
  • Use font-size for text sizes
  • Use text-align for text alignment
